Описание каталога

-CreatePerfomanceCounters
 Если на сервере нужно восстановить 
 набор сборщиков данных (счетчиков производительности Windows)
 для наблюдения за производительностью аппаратного обеспечения,
 процессов сервера приложений 1С
 и состояния СУБД MS SQL Server.
 После запуска скрипта CreatePerfomanceCounters.cmd
 будут созданы три сборщика данных о производительности.
--1C_ExpertTech_counter.xml
 Шаблон сборщика данных из книги Филлипова 1С:Эксперт
--1C_ITS_counters.xml
 Шалон сборщика данных с сайта ИТС
--MSSQLSERVER_counters.xml
 Шаблон сборщика данных рекомендуемый для анализа состояния сервера СУБД

-ExcelPriceImages
 Скрипт для подбора картинок на основе прайса из каталога с картирками.
 Возможно, потребуется установка библиотеки ACE AccessDatabaseEngine_X64.exe 
 
-PRTG
 Бэкап конфигурации системы мониторинга PRTG Network Monitor. 
 Восстановленние из бэкапа:
 а) Остановить службы PRTG.
 б) Вытащить из архива файл конфигурации "PRTG Configuration.dat"
 в) Открыть каталог данных PRTG с помощью скрипта OpenDataFolder.cmd
 г) Положить туда "PRTG Configuration.dat"
 
-TC
 Для Тест-Центра.
--Target_XML
  Каталог для выгрузки целевой конфигурации ИБ в XML.
--TC_XML
  Каталог для выгрузки текущей конфигурации базы Тест-Центра.
--StartTestCenterAgent.ps1
  Скрипт запуска агента Тест-Центра
--Автоматизированное нагрузочное тестирование.odt
  Инструкция по автоматическому встраиванию Тест-центра
  в новые версии конфигурации.
--ВыгрузитьТекущуюБазуТЦвXML.cmd
  Скрипт выгрузки текущей базы Тест-центра в XML
--ВыгрузитьЦелевуюБазувXML.cmd
  Скрипт выгрузки конфигурации целевой ИБ в XML
--ВыгрузкаЗагрузкаДанныхXML.epf
  Обработка из состава конфигурации Тест-Центра.
  Используется для переноса данных ИБ. 
  Может запускаться с параметрами командной строки (!)
--ПодготовкаДанныхТестЦентра.epf
  Разработана для наполнения объектов Тест-Центра,
  которые необходимы для создания сценария тестирования.
  Содержит нано-технологии.
--ПомощникВстраиванияТестЦентра.epf
  Обработка из состава конфигурации Тест-Центра.
  Пример использования описан в "Автоматизированное нагрузочное тестирование.odt".
  
-ПротоколыТестирования
 Протоколы тестирования и образец протакола

-ЦУП
 DT-шник ИБ 1С: Центра Управления Производительностью
